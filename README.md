# 工作流引擎运行流程
```text
             ┌──────────────┐
             │ 加载 JSON 配置 │
             └──────┬───────┘
                    │
        ┌───────────▼────────────┐
        │ 构建 Workflow（包含 Dag）│
        └──────┬────────────┬────┘
               │            │
   ┌───────────▼──┐     ┌───▼──────────────┐
   │ 校验节点与连接 │     │ 构建 NodeType / Op │
   └──────┬────────┘     └───────┬──────────┘
          │                      │
          └────┬────────────────┘
               ▼
      ┌───────────────┐
      │ 拓扑排序（DAG） │
      └──────┬────────┘
             ▼
    ┌────────────────────┐
    │ 顺序执行 Node/Op    │
    │ - 输入收集         │
    │ - 执行 Operation   │
    │ - 输出转发         │
    └─────────┬──────────┘
              ▼
       ┌─────────────┐
```
